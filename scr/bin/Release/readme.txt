【概要】
北海道を落としたらどうなるんだろう？　そんな疑問にお答えします。

このスクリーンセーバーは各都道府県を落としたときの様子を観察することが
できます。


【インストール】
お好きなフォルダに解凍してください。

hokkaido.scr を右クリックして [インストール] を選びます。
[設定] ボタンより落ちる順番を設定することができます。

※Flash Player 9.0 以上と .NET Framework 2.0 以上がインストールされて
  いる必要があります。


【アンインストール】
ファイルごと削除して下さい。
